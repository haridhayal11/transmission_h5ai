# transmission_h5ai
A script to install and setup transmission and h5ai file index. 

## Usage

First, get the script and make it executable

```bash
curl -O https://raw.githubusercontent.com/haridhayal11/transmission_h5ai/master/setup.sh
chmod +x setup.sh
```

Then run it

```sh
sudo bash setup.sh
```
